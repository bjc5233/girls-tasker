# girls-tasker


## 说明
在桌面小工具中展示一个萌娘（或者其他的图片），每次点击更换图片
* 安装tasker，以及插件zoom
* 导入tasker项目girls.tsk.xml
* 修改图片文件夹路径
* 导入zoom项目girls.ztl.xml，并在桌面添加小工具
